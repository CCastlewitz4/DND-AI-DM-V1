This repository is version 1 to a long standing project I've been working on. It is buggy and broken but works "mostly" with some features not working 100%
It utilizes the Obsidian file storage system, which can be downloaded here: https://obsidian.md/
Future systems will most likely continue implementing Obsidian 
Currntly utilizes Llama3.1:8b-instruct-q4_0 LLM for the AI system: https://www.llama.com/ OR https://huggingface.co/meta-llama/Llama-3.1-8B-Instruct

Goals for future changes:
Reduce number of files to make readability simpler
Possibly remove the other system generator - reduce complexity

System goals:
1. Recall events memory correctly and reference them in longstanding exchanges/story telling
2. Produce semi decent images, whether they be created from the system or using a free 3rd party software
3. interface similarly to a game - easily create new games while having others saved, new characters, etc
4. Ensure that the system remains fun and easy to use


If I can achieve these goals, I will be satisfied with the system. For now, I will continue working to improve and perfect the system
